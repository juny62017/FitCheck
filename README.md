# FitCheck

A soft and aesthetic BMI calculator built  by meeeeee!!!
## Description

FitCheck is a simple website  BMI(body mass index) calculator that helps 
users calculate their Body Mass Index using height and weight details

## Features

- BMI calculation
- Height 
- Weight
- Age 
- Gender selection
- Health category detection
- BMI progress meter
- Reset function

## How To Use / Setup

- Click on the link
- Enter your height and weight
- Click the Calculate BMI button
- View your BMI result and health category 
- Use the Reset button to clear data

MAKE SURE TO CLICK ON RESET BEFORE PUTTING OTHER VALUES
